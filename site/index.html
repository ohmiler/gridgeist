04 / System rules

Restraint creates
recognition.

Not a preset. A way to make decisions under pressure.

G

Grid

Align meaning, not just boxes. Break structure only to create deliberate emphasis.

T

Type

Define roles. Use weight and spacing before inventing more sizes.

E

Evidence

Let real code, data, lessons, states, and workflows carry visual weight.

R

Response

Recompose small screens around priority. Never shrink desktop and call it done.
