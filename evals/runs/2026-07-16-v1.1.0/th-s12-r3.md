ยังแก้โค้ดให้ไม่ได้ในรอบนี้ เพราะไม่มี target fixture หรือไฟล์ที่มีปุ่ม Save ให้ตรวจแก้ และผมจะไม่สมมติโครงสร้างของโปรเจกต์

โปรดส่งไฟล์คอมโพเนนต์/Click handler ที่เกี่ยวข้อง (พร้อมคำสั่งทดสอบ ถ้ามี) แล้วผมจะเปลี่ยนเฉพาะพฤติกรรมให้การคลิกหนึ่งครั้ง Submit ได้ไม่เกินหนึ่งครั้ง พร้อมเพิ่มหรือปรับการทดสอบที่ยืนยันการป้องกัน Submit ซ้ำ โดยจะไม่เปลี่ยน markup, styling, layout หรือ interaction อื่นของหน้าเว็บ
