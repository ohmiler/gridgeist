# Commonroom direction-alignment fixture

Commonroom is a fictional neighborhood workshop directory for adults who want to
compare upcoming local sessions and reserve a place. The primary task is to find a
suitable workshop by topic and availability, inspect its details, and reserve one
seat.

## Product facts

- The fixture content is fictional and must remain labeled as such.
- Residents need the workshop topic, host, date, time, location, accessibility
  note, price, and remaining capacity before deciding.
- The topic and availability filters, details dialog, reservation action, focus
  return, and visible reservation confirmation are required behavior.
- The three workshop records and their factual content must be preserved.
- No customer research, conversion data, testimonials, or approved visual direction
  is supplied.

## Run locally

Serve this directory with any static HTTP server and open `index.html`. Evaluation
runs that implement a redesign should operate on a disposable copy so later runs
start from the same baseline.
